# Лабиринт сокровищ

Текстовая приключенческая игра-лабиринт, написанная на чистом Python без внешних зависимостей.

## Описание

"Лабиринт сокровищ" — это интерактивная текстовая игра, в которой игрок исследует лабиринт, решает загадки, собирает предметы и пытается добраться до сокровищницы. Игра включает:

- **Исследование комнат**: перемещение по лабиринту с описаниями локаций
- **Загадки**: решение головоломок для получения наград
- **Предметы**: сбор и использование различных предметов
- **Случайные события**: неожиданные происшествия во время путешествия
- **Ловушки**: опасности, которые могут привести к потере предметов или поражению

## Установка

### Требования

- Python 3.10 или выше
- Poetry (для управления зависимостями)

### Шаги установки

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd project1_kalenskiy_egor_M25-555
```

2. Установите зависимости:
```bash
poetry install
```
или
```bash
make install
```

## Запуск

Запустите игру командой:
```bash
poetry run project
```
или
```bash
make project
```

## Команды игры

- `go <direction>` или `north/south/east/west` — перемещение в указанном направлении
- `look` — осмотреть текущую комнату
- `take <item>` — поднять предмет
- `use <item>` — использовать предмет из инвентаря
- `inventory` или `inv` — показать инвентарь
- `solve` — попытаться решить загадку в комнате
- `help` — показать список команд
- `quit` или `exit` — выйти из игры


## Технические детали

- **Язык**: Python 3.10+
- **Зависимости**: Только стандартная библиотека (модуль `math`)
- **Линтер**: Ruff
- **Стиль кода**: PEP 8, snake_case для функций, UPPER_SNAKE_CASE для констант
